### git 本机
    git add ./                  ## 添加上传文件
    git commit -m "备注"          ## 提交到本地仓库
    git push                    ## 提交到远程仓库【github】
    git clone                   ## 从远程仓库拉取一个新的项目
    git pull                    ## 从远程仓库更新到本地仓库
